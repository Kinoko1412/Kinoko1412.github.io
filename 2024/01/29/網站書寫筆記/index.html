  2. 分數 (Fraction)

    • 表達式：\frac{分子}{分母}
    • 例子：$\frac{a}{b}$ 渲染為 ab\frac{a}{b}ba​

  3. 上標和下標 (Superscript and Subscript)

    • 上標：a^b
    • 下標：a_b
    • 例子：$x^2$ 渲染為 x2x^2x2，$x_i$ 渲染為 xix_ixi​

  4. 根號 (Square Root)

    • 表達式：\sqrt{表達式}
    • 例子：$\sqrt{x}$ 渲染為 x\sqrt{x}x​

  5. 希臘字母 (Greek Letters)

    • 小寫：\alpha, \beta, \gamma, …
    • 大寫：\Alpha, \Beta, \Gamma, …
    • 例子：$\alpha$ 渲染為 α\alphaα，
    • $\Gamma$ 渲染為Γ\GammaΓ

  6. 積分 (Integral)

    • 表達式：\int_{下限}^{上限}
    • 例子：$\int_{a}^{b} x dx$ 渲染為 ∫abx dx\int_{a}^{b} x \, dx∫ab​xdx

  7. 求和 (Summation)

    • 表達式：\sum_{下限}^{上限}
    • 例子：$\sum_{i=1}^{n} i$ 渲染為 ∑i=1ni\sum_{i=1}^{n} i∑i=1n​i

    • 正弦函數：$\sin(\theta)$ 渲染為 sin⁡(θ)\sin(\theta)sin(θ)

  2. 排列組合 (Permutations and Combinations)

    • 排列：$_nP_r = \frac{n!}{(n-r)!}$ 渲染為 _nPr=n!(n−r)!\_nP_r = \frac{n!}{(n-r)!}_nPr​=(n−r)!n!​

hexo常用指令

Hexo 常用指令摘要

指令列表

1. 啟動本地伺服器

  • 完整指令：hexo server
  • 縮寫：hexo s
  • 功能：啟動本地伺服器，預覽網站。

2. 生成靜態文件

  • 完整指令：hexo generate
  • 縮寫：hexo g
  • 功能：生成靜態網頁文件。

3. 部署網站

  • 完整指令：hexo deploy
  • 縮寫：hexo d
  • 功能：將生成的靜態網頁部署到遠端伺服器。

4. 創建新文章

  • 完整指令：hexo new [layout] <title>
  • 縮寫：無
  • 功能：根據指定的布局（layout）創建一篇新文章。

5. 清理快取文件

  • 完整指令：hexo clean
  • 縮寫：無
  • 功能：清理快取文件和已生成的靜態文件。

6. 列出網站資訊

  • 完整指令：hexo list <type>
  • 縮寫：無
  • 功能：列出網站的資訊，如文章、分類等。

7. 版本資訊

  • 完整指令：hexo version
  • 縮寫：無
  • 功能：顯示 Hexo 的版本資訊。

組合指令

  • 啟動本地伺服器並生成靜態文件：hexo s -g
  • 生成靜態文件並部署：hexo g -d

以上是 Hexo 的一些常用指令，適用於日常的博客管理和維護工作。
